# Wedding Project

Персонализированное свадебное приглашение. Сайт читает query-параметр `guest`, показывает индивидуальное обращение для гостей, программу дня, локации, дресс-код, контакты координатора и RSVP.

## Стек

- React 19
- TypeScript
- Webpack 5 и webpack-dev-server
- ESLint и Prettier
- Yarn
- GSAP и `@gsap/react`
- Motion
- Three.js
- Lucide React
- React Yandex Maps
- Vitest, Testing Library и Playwright

## Команды

```bash
yarn install
yarn dev
yarn build
yarn test:unit
yarn lint
```
